# Student Management System (Java)

This is a console-based Java application developed to manage student records and practice Java fundamentals and object-oriented programming concepts.

## Features
- Add student details
- View all students
- Search student by ID
- Calculate average marks
- Display pass/fail status

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## Purpose
This project was developed as part of my academic learning to improve Java programming skills.
